pip install -U kaleido  هو اسم المكتبة التي تريد تثبيتها. kaleido هي مكتبة تُستخدم لتصدير الرسوم البيانية من مكتبة Plotly إلى تنسيقات صور مثل PNG وJPEG

openpyxl:
تُستخدم لقراءة وكتابة ملفات Excel (بتنسيق XLSX). تعتبر مفيدة عند العمل مع البيانات المخزنة في جداول Excel.
plotly:
مكتبة لرسم الرسوم البيانية التفاعلية. تُستخدم لإنشاء مخططات ورسوم بيانية جميلة يمكن تضمينها في تطبيقات الويب.
mysql-connector-python:
مكتبة تُستخدم للتواصل مع قواعد بيانات MySQL. تسمح لك بالاتصال بقاعدة بيانات MySQL وتنفيذ استعلامات SQL.
bcrypt:pip install bcrypt
مكتبة تُستخدم لتشفير كلمات المرور. توفر طرقًا لتشفير وفك تشفير البيانات بطريقة آمنة.
